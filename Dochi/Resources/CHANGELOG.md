# Changelog

## v1.0.0

### 새로운 기능
- 멀티 LLM 제공자 지원 (OpenAI, Anthropic, Z.AI)
- Apple STT를 활용한 음성 입력
- Supertonic TTS를 활용한 자연스러운 한국어 음성 출력
- 웨이크워드 감지 기능
- 연속 대화 모드
- 대화 히스토리 저장 및 관리
- 시스템 프롬프트 커스터마이징 (system.md)
- 사용자 기억 자동 저장 (memory.md)
- 메모리 자동 압축 기능

### 개선사항
- 테스트 가능한 구조로 리팩토링 (프로토콜 기반 DI)
- 단위 테스트 인프라 구축
