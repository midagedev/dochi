# Dochi - 컨셉 문서

## 정의

Dochi는 음성 기반 AI 어시스턴트 macOS 앱입니다.

## 이름

"도치"는 고슴도치에서 따왔습니다. 부르기 쉬운 이름이 필요했습니다.

## 주요 특징

### 로컬 우선 (Local-First)

- **TTS**: Supertonic ONNX 모델 (로컬)
- **STT**: Apple Speech Framework (온디바이스 가능)
- **데이터**: 모든 대화/기억이 로컬에 저장
- **LLM**: API 키는 사용자가 직접 관리

### 멀티 LLM

OpenAI, Anthropic, Z.AI 중 선택 가능. 상황에 따라 바꿔 쓸 수 있습니다.

### 컨텍스트 관리

- `system.md`: 페르소나 정의
- `memory.md`: 장기 기억 자동 축적
- 대화 종료 시 자동 분석/저장

### 연속 대화

웨이크워드("도치야") → 대화 → 자연스러운 종료 흐름.

## 사용 시나리오

### 가정용

- 아이들 대화 상대 (프로필별 페르소나 설정 가능)
- 음성 중심 인터랙션

### 개발자용

- LLM 실험
- 커스텀 페르소나 테스트
- (향후) 에이전트 확장

## 설계 원칙

1. **단순하게** - 기본 사용은 쉽게
2. **투명하게** - 데이터 로컬 저장, 오픈소스
3. **확장 가능하게** - 프로토콜 기반 구조
